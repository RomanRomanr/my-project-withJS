

# 🪑 Mebleria

## 📌 Про проєкт

Mebleria — це вебсайт для пошуку та замовлення меблів і елементів інтер’єру для дому. Проєкт створений для того, щоб користувачі могли швидко, зручно та ефективно обирати потрібні товари завдяки простому інтерфейсу та зрозумілому функціоналу.

 ### 🎯 Основна ідея

 Надати користувачам можливість:

#### 1. легко знаходити меблі

#### 2. переглядати детальну інформацію про товари

#### 3. швидко оформлювати замовлення

## 🚀 Запуск проєкту
### Підготовка до роботи

 Переконайся, що на комп'ютері встановлено LTS-версію Node.js.
   [Скачай та встанови](https://nodejs.org/en/) її якщо необхідно.

1. Клонування репозиторію

git clone <your-repo-link>
cd <project-folder>

2. Встановлення залежностей

npm install

3. Запуск у режимі розробки

npm run dev

4. Збірка проєкту

npm run build

5. Перегляд production-версії

npm run preview

## 📂 Структура проєкту

src/ — основний код проєкту

public/ — статичні ресурси

dist/ — зібрана версія проєкту

## 📎 Додаткова інформація

Проєкт орієнтований на користувачів, які шукають сучасні рішення для облаштування інтер’єру. Основний акцент зроблено на зручність використання, швидкість роботи та приємний користувацький досвід.
